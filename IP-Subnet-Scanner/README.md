# PowerShell IP Subnet Calculator with GUI

A professional networking utility featuring a Graphical User Interface (GUI) built with PowerShell. This tool simplifies complex subnetting calculations by providing a visual interface for network administrators.

## Technical Features
* Graphical User Interface: User-friendly input fields and buttons for seamless interaction.
* Automated Calculations: Calculates Network Address, Broadcast Address, and Subnet Mask.
* Host Management: Displays the first usable host, last usable host, and total number of hosts.
* Visual Output: Clear presentation of results within the application window.

## Project Structure
* src/: Contains the source code (Subnetter.ps1) including the GUI XAML/Windows Forms code.
* README.md: Technical documentation and usage instructions.

## Requirements
* Windows PowerShell 5.1 or PowerShell 7+
* .NET Framework (standard on Windows)

## How to Use
1. Download the "Subnetter.ps1" file.
2. Right-click the file and select "Run with PowerShell".
3. Enter the IP address and subnet details in the GUI window and click Calculate.
